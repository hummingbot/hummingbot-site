# Release Notes - Version 1.17.0

*Released on June 26, 2023*

We are very excited to ship the June 2023 release of Hummingbot (v1.17.0) today!

This release marks the launch of the Hummingbot Dashboard community project, which now includes the Bot Orchestration feature giving users the ability to manage bots from a single dashboard. We added support for market orders on Gate.io and significant enhancements to the Position Executor. Additionally, the Candles feature has been updated, and a new DEX connector named Polkadex has been introduced.

## Market Data Collector

Hummingbot now has the functionality that allows for the collection of market data in the background from the markets that are initialized in the running strategy. The market data collection feature is added as a new config variable in the `conf_client.yml` file. Users can configure the following parameters:

- Enabled: Set to True/False to enable or disable market data collection.
- Interval: Specify the interval in seconds for collecting data (e.g. 60 seconds).
- Depth: Set the number of levels of the order book to collect (e.g. 20 levels).

The data collected includes `exchange`, `trading pair`, `timestamp`, `mid-price`, `best ask`, `best bid`, and `order book`. The data is stored in a new table in the sqlite DB called `MarketData`

**Pull Requests:** [#6364](https://github.com/hummingbot/hummingbot/pull/6364)

## Position Executor

This release includes the refactoring of the Position Rxecutor in Hummingbot. The key changes include:

- Creation of a smart components base class.
- Introduction of tests for the smart components base.
- Refactoring of the Position executor, which included:
    a. Spot and Perpetual trading integration.
    b. Implementation of a Trailing Stop feature.
    c. Improvements to the control logic.
- Addition of tests for the Position executor.
- Refactoring of strategies that utilize the directional framework‚Äã
