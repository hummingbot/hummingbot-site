### How to Connect

From inside the Hummingbot client, run `connect btc_markets`:

```
>>> connect btc_markets

Enter your btc_markets API key >>>
Enter your btc_markets secret key >>>
```

If connection is successful:

```
You are now connected to btc_markets
```
